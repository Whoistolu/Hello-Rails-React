# HELLO RAILS REACT APP

 A project using the `react-rails` gem to build React components as a part of Rails JavaScript and serve it in a component, react_component, used in a regular ERB file.

## Built with

- React
- Ruby on Rails
- Visual Studio Code
- Lots of love :heart:
